# Adjustable DC Power Control using NE555 & TIP122

## 📌 Overview
This project is an adjustable DC control circuit designed using KiCad. 
The design includes a bridge rectifier, filtering capacitor, NE555 timer IC for PWM generation, and TIP122 transistor for load driving.

## 🔧 Features
- AC to DC rectification using 1N4007 diodes
- 1000uF smoothing capacitor
- NE555 timer in PWM configuration
- Adjustable output using potentiometer
- TIP122 transistor for high current load control
- LED power indicator
- Screw terminal input/output
- Full PCB layout & 3D model

## ⚙️ Working Principle
AC input is rectified and filtered.
The NE555 timer generates PWM signal.
The PWM drives the TIP122 transistor.
Output voltage/current can be adjusted using potentiometer.
